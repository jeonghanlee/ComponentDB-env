# Configuration Env

Configuration Environment for ComponentDB at <https://github.com/AdvancedPhotonSource/ComponentDB>

## Requirements

```bash
make init
```

* Package, patch, and environment

```bash
make install.pkgs
```

```bash
make patch.apply
make build
make deploy.cdb
```
